How to Use

  • Adjust Cell Size: Use your mouse wheel over the maze to make cells larger or smaller.
  • Resize the Maze: Drag the bottom-right corner of the maze to change its dimensions.
  • Generate a New Maze: Click the reload button or enter a number in the seed input to create a new random maze.
  • Download: Save your maze as an SVG or PNG file using the download buttons.
  • Trace the Path: Click and drag to trace the path from start to end.
